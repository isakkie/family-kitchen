# 家庭厨房

一个家庭菜谱知识库原型，当前版本是纯前端本地应用，数据保存在当前浏览器的本地存储中。

## 已包含

- 菜谱库：查看、搜索、按标签筛选。
- 导入：粘贴菜谱正文后识别标题、用料、步骤、小贴士。
- 批量导入：支持 JSON、多个正文分隔导入，或一行一个链接先占位。
- 备份导出：将当前本地数据导出为 JSON。
- 家庭密码页：默认密码 `kitchen2026`，进入后可手动锁定。
- PWA：部署到网址后可在 iPhone / 安卓添加到桌面。
- 编辑：导入后可快速修改用料、步骤、标签和家庭备注。
- 冰箱：记录现有食材。
- 保鲜天数：新增食材时默认 3 天，也可以直接填 `1`、`5`、`10天`，系统会自动换算成到期日。
- 推荐你做：根据现有食材计算匹配度，显示缺少的主料。
- 采购：从菜谱或缺失食材生成购物清单。
- 菜单：安排一周菜单。

## 打开方式

当前本地预览地址：

```text
http://127.0.0.1:4174/index.html
```

也可以直接打开 `index.html` 使用。

## 家庭访问

当前默认家庭密码是：

```text
kitchen2026
```

这个密码页适合家庭试用和阻挡普通访客。因为当前应用是纯前端静态网页，它不是强安全方案；正式长期使用时，建议配合部署平台的访问控制，或后续接入家庭账号。

当前数据保存在各自浏览器本地。也就是说，你手机里的菜谱不会自动同步到你的其他设备；需要用“导出备份 / JSON 文件导入”同步，或后续接入云端数据库。
